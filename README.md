# chn-site-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面是对特定时期、特定主题或特定内容片段的静态保存，旨在为研究、参考或回顾提供一份未经修改的原始副本。

## 项目简介

本项目是一个通用型 HTML 页面归档仓库。仓库中收录的页面均以独立文件形式存在，不依赖于任何特定的服务器环境或数据库。每个页面均保留了其被收录时的原始结构和内容，未进行功能性修改或美化。

## 目录说明

- `archive/`：存放所有归档的 HTML 页面文件。每个文件以描述性名称命名，便于识别内容主题。
- `index.html`：仓库根目录下的索引页面，提供归档页面的导航与简要说明。
- `assets/`：可选目录，存放归档页面引用的静态资源（如图片、样式表、脚本等），确保页面独立可展示。

## 页面归档说明

- 每个归档页面均为完整独立的 HTML 文件，可直接在浏览器中打开查看。
- 归档内容仅做静态保存，不保证与原始来源完全一致，也不代表当前状态。
- 归档文件不包含任何跟踪代码、广告或外部依赖，除非原始页面本身包含。
- 文件命名遵循 `{分类}-{描述}.html` 的格式，具体含义可在索引页面中查阅。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `index.html` 或直接进入 `archive/` 目录，选择感兴趣的页面。
3. 在浏览器中打开 HTML 文件即可浏览归档内容。

## 维护说明

- 本仓库不定期更新，新增归档页面。
- 维护者会尽力确保归档文件的完整性和可访问性，但不对文件内容的准确性或时效性作额外承诺。
- 如需提交新的归档内容，请通过 Issues 或 Pull Requests 联系维护者。所有提交需符合归档主题且不违反相关法律法规。
- 本仓库不提供删除或修改已归档页面的功能，以保持归档的原始性。
